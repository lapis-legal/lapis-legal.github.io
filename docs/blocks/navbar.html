Note that since we show multiple navbar examples on docs, offcanvas navigation, offcanvas info and modals are disabled on the examples, but their codes are intact and will work properly on the template.
Disable dropdown hover by adding .hover-none class to .navbar on Include classList.
Disable carets on dropdown parents by adding .caret-none class to .navbar on Include classList.
